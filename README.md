## Ход работы

После открытия файла в IDA Pro была исследована функция проверки серийного номера `sub_4017BD`.

В ходе анализа установлено, что программа ожидает ввод ключа следующего формата:

```text
XXXXXXXX-XXXXXXXX-XXXXXXXX-XXXXXXXX
```

Каждый блок представляет собой шестнадцатеричное число. После проверки формата введённые данные преобразуются в четыре 32-битных значения:

```c
v5 = strtoul(...);
v6 = strtoul(...);
v7 = strtoul(...);
v8 = strtoul(...);
```

Далее программа выполняет серию арифметических и логических преобразований в цикле длиной `v8` итераций:

```c
v12 = (v6 ^ v5) + 986598842;
v13 = (v12 ^ (v7 + 946341097)) + v10;
v14 = v13 ^ (v12 >> 3);

v5 = (v14 + 1670772768) ^ 0xE93E7D9A;
v6 = (v13 + (v10 ^ v6)) ^ 0x4E192BBA;
v7 = (v12 ^ (v14 + 1670772768)) + 1594596065;
```

После завершения цикла производится сравнение полученных значений с константами, расположенными в секции данных программы:

```c
if (
    dword_404024 == v5 &&
    dword_404020 == v6 &&
    dword_40401C == v7 &&
    dword_404018 == v8
)
```
